# Vogue Vault - Headless E-Commerce

**Role:** Frontend Developer  
**Tech Stack:** Next.js / Tailwind CSS / Shopify / Three.js / Framer Motion  

## Overview
**Vogue Vault** is a **headless e-commerce platform** that delivers a premium shopping experience with:  
- **3D product previews:** Interactive models for an immersive feel  
- **Seamless page transitions:** Smooth navigation across the site  
- **Fast, modern frontend:** Optimized performance with Next.js and Tailwind  
- **Shopify integration:** Headless commerce for flexible backend management
